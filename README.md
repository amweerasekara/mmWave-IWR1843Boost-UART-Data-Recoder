# mmWave-IWR1843Boost-UART-Data-Recorder
This repository contains a python script to record UART data from TI IWR1843Boost (SDK3)

## Install Requirments

```bash
pip install -r requirements.txt
```

## Usage


```bash
python record_dat.py
```
